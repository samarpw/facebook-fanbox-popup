== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Facebook FanBox Popup'
3. Activate 'Facebook FanBox Popup' from your Plugins page.
4. Visit 'Setting > 'Facebook FanBox Popup'.
